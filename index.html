Web-firewall

The firewall project for mitigating HTTP Layer 7 DDoS attack through PHP.

Download as .zip Download as .tar.gz View on GitHub

Welcome to the Web-Firewall page.

The Web-Firewall is a lightweight project for protecting common home-hosted servers, using iptables or cloudflare if available. Currently supported: Debian GNU/Linux Apache 2.24 PHP 5.6.7+dfsg-1

To download, do.

$ git clone https://GitHub.com/HoleInTheWall/Web-Firewall
